Object-Oriented Programming (OOP) with Python
---
This repository contains code documentation, exercises, and case studies dedicated to mastering the Object-Oriented Programming (OOP) paradigm using Python. This project was built to understand how to structure code that is modular, maintainable, and reusable.

📌 Project Status: ✅ Completed / Maintained
The implementation of core concepts and practical OOP exercises within this module has been completed. This repository serves as an academic portfolio and a reference for object-oriented programming logic that will be continuously maintained.

🚀 Implemented OOP Concepts
Within this repository, there are real-world implementation examples of the main pillars of OOP, including:

Class & Object: Defining the blueprint of objects, attributes, and methods.

Constructor & Methods: Utilizing __init__ for object initialisation and creating specific methods.

Encapsulation: Restricting direct access to attributes/methods using private (__) or protected (_) variables.

Inheritance: Deriving traits, attributes, and methods from a superclass (parent class) to a subclass (child class).

Polymorphism: Implementing functions or methods with the same name but different behaviours across different objects.
